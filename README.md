
# todoApp (React)

Сайт для добавления, удаления и пометки сделанных задач.

На данный момент реализовано:

- добавление задачи
- переключение состояния задачи (выполнено/не выполнено)
- удаление
- три страницы (все задачи/выполненные/не выполненные)
